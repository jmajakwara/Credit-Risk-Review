Cure Rate Modelling: State of the Art in Credit Risk

This repository contains the R code that generated the results presented in the Tables in the manuscript: Cure Rate Modelling: State of the Art in Credit Risk 

The file https://jmajakwara.github.io/Credit-Risk-Review/Cure_Rate_Credit_Risk.html is an interactive table that lists all the articles used in the preparation for writing the article.  

The files are organised as follows. BoxCox_p2p_R provides the code for generating results of Prosper peer-to-peer lending data using Box-Cox transformation and Mixture cure rate models. The data can be downloaded from https://www.kaggle.com/datasets/henryokam/prosper-loan-data and was cleaned using the codes provided by https://github.com/jmajakwara/Credit-risk. MC_logn_BoxCox_1.0.R provides the code to generate simulation results by changing the values of alpha and n (sample size).
